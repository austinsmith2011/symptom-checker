# Symptom Checker

AI-powered symptom analysis tool that helps identify potential health conditions, treatment options, and urgency levels.

**This is not a substitute for professional medical advice.** Always consult a qualified healthcare provider for medical concerns.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Add your OpenAI API key to `.env`:
   ```
   OPENAI_API_KEY=sk-your-key-here
   ```

3. Start the server:
   ```bash
   npm start
   ```

4. Open http://localhost:3000 in your browser.

## Deployment

Deployed on Render at `symptom-checker.appsthataustinmade.com`. Set the `OPENAI_API_KEY` environment variable in the Render dashboard.
